# Food Recognition and Calorie Estimation

## Objective

Develop a model that recognizes food items from images and estimates calorie content.

## Approach

* Used Food-101 Tiny dataset
* Applied Transfer Learning (MobileNetV2)
* Built CNN model for classification
* Mapped food items to calories manually

## Output

* Predicts food item from image
* Displays estimated calories

## Technologies

* Python
* TensorFlow / Keras
* OpenCV (optional)

## Result

Model successfully classifies food items and estimates calorie values.
